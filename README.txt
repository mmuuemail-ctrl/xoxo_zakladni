xoxo – Web Data Updater (MVP)
=============================

Jak spustit (Windows 10/11):
1) Nainstaluj Node.js (https://nodejs.org – LTS).
2) V této složce (xoxo) otevři PowerShell / CMD:
   npm install
   npm start

Použití:
- Zadej URL a klikni Otevřít stránku.
- Klikni "Vybrat prvek", na stránce klikni na požadovaný text/číslo (zvýrazní se).
- Vyplň Název + Poznámka a dej "Přidat řádek".
- "Aktualizovat teď" načte aktuální hodnoty...
- Export CSV uloží tabulku pro Excel/Google Sheets.
- Trial běží 7 dní od prvního spuštění.

Poznámky:
- Přihlášení: přihlas se ve vloženém okně stránky.
- Výběr prvku používá CSS selektor...
